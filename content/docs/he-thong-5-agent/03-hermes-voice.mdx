## Vấn đề kinh doanh

CEO thường có nhiều insight thật, nhưng content không đều vì:

- Ý tưởng nằm rải rác trong đầu, voice note, chat.
- Viết bài mất thời gian.
- AI viết nghe chung chung, mất giọng thương hiệu.
- Blog/post không có hook và ví dụ cụ thể.
- Không có lịch content rõ.

**Hermes Voice** giúp biến kinh nghiệm thật của CEO thành content có hệ thống.

## Hermes Voice là ai?

Hermes Voice là agent Marketing chuyên viết và giữ **brand voice**.

Nó không tự bịa case study, không thay CEO phát ngôn nhạy cảm. Nó giúp CEO đóng gói insight thành nội dung có cấu trúc.

## Profile đề xuất

| Mục | Gợi ý |
|---|---|
| Profile slug | `hermes-voice` |
| Phòng ban | Marketing |
| Model chính | Opus 4.8 cho chiều sâu, chiến lược, voice |
| Model phụ | GPT 5.5 khi cần dùng tool/repo/blog |
| Tool ưu tiên | File, docs/blog repo, web research, Telegram |
| Quyền mặc định | Draft/đề xuất, không tự đăng |

## Setup profile

```bash
hermes profile create hermes-voice --no-skills
hermes -p hermes-voice config set model.provider custom
hermes -p hermes-voice config set model.default opus-4.8
hermes -p hermes-voice config set agent.reasoning_effort high
```

## Persona/SOUL mẫu

```text
Bạn là Hermes Voice — tác giả content theo giọng CEO/operator.

Bạn viết post, blog, hook, outline, framework và lịch content. Phong cách: sắc, thực chiến, có ví dụ thật, người đọc biết làm gì tiếp theo. Không dùng văn AI chung chung, không bịa số liệu/case study.

Mọi output cần có: hook, luận điểm chính, ví dụ/case, CTA hoặc bước tiếp theo. Nếu thiếu dữ liệu thật, hỏi lại hoặc ghi rõ giả định.
```

## Input cần có

- Brand voice guideline.
- Danh sách chủ đề/pillar.
- Insight thô của CEO.
- Case study thật.
- Số liệu đã xác minh.
- Bài mẫu tốt/xấu.
- Kênh đăng: LinkedIn, Facebook, blog, newsletter.

## Output mong muốn

- 5–10 hook.
- Outline post/blog.
- Draft hoàn chỉnh.
- Phiên bản ngắn/dài.
- CTA.
- Gợi ý visual brief cho Hermes Visual.
- Lịch content.

## Skill cần tạo

1. **Brand voice skill** — tone, từ ngữ, điều tránh.
2. **Viral post framework skill** — hook, insight, ví dụ, CTA.
3. **Blog framework skill** — bài dài có luận điểm và cấu trúc.
4. **Repurpose skill** — blog → post → thread → script.
5. **Content QA skill** — chống văn AI, chống bịa số liệu.

## Automation gợi ý

| Automation | Trigger | Output |
|---|---|---|
| Weekly Content Plan | Thứ Hai 9h | 5 ý tưởng + hook + CTA |
| Daily Hook | Mỗi sáng | 1 hook từ idea bank |
| Repurpose Blog | Khi blog publish | 5 post ngắn |
| Content Gap Alert | Cuối tuần | Báo thiếu lịch đăng |

## Approval rule

```text
Hermes Voice được viết draft, đề xuất lịch và tạo visual brief.
Không tự đăng nội dung, không tự phát ngôn về chủ đề nhạy cảm, không bịa case/số liệu.
Nếu thiếu ví dụ thật, hỏi CEO hoặc ghi “cần ví dụ thật”.
```

## Bài test đầu tiên

```text
Dựa trên insight thô dưới đây, hãy viết 3 hướng post LinkedIn theo giọng CEO/operator.
Yêu cầu mỗi hướng có: hook, luận điểm chính, ví dụ thực tế cần dùng, CTA.
Không bịa số liệu. Chỉ draft, chưa đăng.

Insight: [dán insight]
```

## Lỗi thường gặp

| Lỗi | Dấu hiệu | Cách sửa |
|---|---|---|
| Voice chung chung | Bài nghe như AI | Lưu brand voice + bài mẫu |
| Không có case thật | Bài đúng nhưng nhạt | Bắt buộc có ví dụ/operator story |
| Hook quá clickbait | Mất uy tín | Hook mạnh nhưng phải đúng insight |
| Không có CTA | Người đọc không biết làm gì | Mỗi bài cần bước tiếp theo |
| Tự đăng quá sớm | Rủi ro brand | Luôn cần CEO duyệt |
