## Khóa này dành cho ai?

**Dành cho:**
- Solo CEO, founder, chủ doanh nghiệp nhỏ, operator, agency owner.
- Người đang lặp lại nhiều việc: báo giá, email, content, nghiên cứu, tổng hợp dữ liệu, chăm sóc khách hàng.
- Người muốn vận hành như có một phòng ban — nhưng không có ngân sách thuê đủ người.

**Không dành cho:**
- Người chỉ muốn học ChatGPT để viết nhanh hơn.
- Người không sẵn sàng thay đổi cách vận hành.
- Doanh nghiệp lớn có team IT riêng — nên dùng giải pháp enterprise.

---

## Nếu tôi không biết code thì sao?

**Không sao.** Khóa này thiết kế cho CEO, không phải cho developer.

Bạn cần biết:
- Gõ lệnh terminal cơ bản (copy-paste được).
- Dùng Telegram.
- Hiển hiểu khái niệm: agent, skill, cron, memory — khóa sẽ giải thích bằng ngôn ngữ vận hành.

Phần setup kỹ thuật có thể nhờ người rành làm trong 30–60 phút. Sau đó, CEO vận hành qua Telegram.

## Hermes khác gì ChatGPT / Zapier / Make?

| Tiêu chí | ChatGPT | Zapier / Make | Hermes |
|---|---|---|---|
| Cách dùng | Hỏi-đáp từng lần | Kết nối app qua app | Giao việc như cho nhân viên |
| Trí nhớ | Mất ngữ cảnh | Không có | Memory, session, skill |
| Hành động | Chủ yếu text | Kết nối API | Tool, file, web, cron, webhook |
| Việc lặp | Nhắc lại quy trình | Cần build flow | Ghi thành skill, dùng lại |
| Vai trò CEO | Vẫn thao tác | Setup flow phức tạp | Ra lệnh, duyệt, kiểm soát |

**Tóm lại:** ChatGPT trả lời. Zapier nối app. Hermes vận hành như nhân viên AI.

## Sau 4 tuần tôi có gì cụ thể?

| Tuần | Output |
|---|---|
| Tuần 1 | Top 3 việc nên agent hóa + Agent Brief đầu tiên |
| Tuần 2 | 1 skill chạy được + Approval Rule |
| Tuần 3 | 1 cron Daily Brief chạy hằng ngày |
| Tuần 4 | 1–2 agent chạy việc thật + Weekly Review đầu tiên |

**Kết quả tối thiểu:** 1 agent chạy hằng ngày.
**Kết quả tốt:** 2 agent + 1 cron + 1 skill + bộ template vận hành.

## Mất bao nhiêu thời gian mỗi ngày?

| Nhóm | Thời gian/ngày | Kết quả |
|---|---|---|
| Bận | 45–60 phút | Có agent đơn giản |
| **Chuẩn** | **60–90 phút** | **Có agent chạy thật** |
| Tăng tốc | 90–120 phút | Có 2 agent + automation |

---

## Cần chuẩn bị gì trước khi học?

| Cần có | Ghi chú |
|---|---|
| Máy tính Linux/macOS hoặc WSL | Để cài Hermes |
| Telegram | Để giao tiếp với agent |
| API key từ provider AI | OpenRouter hoặc LLMGate |
| 20–50 việc lặp trong business | Để chọn use case ưu tiên |
| 60–90 phút/ngày | Để thực hành |

## Chi phí chạy Hermes bao nhiêu?

Phụ thuộc vào model và tần suất dùng.

| Mục | Chi phí ước tính/tháng |
|---|---|
| VPS (nếu cần 24/7) | 200k–500k VNĐ |
| API key (model AI) | 200k–2 triệu VNĐ (tùy用量) |
| **Tổng** | **400k–2.5 triệu VNĐ/tháng** |

So với thuê 1 nhân sự part-time: 5–10 triệu/tháng.

---

## Nếu agent làm sai怎么办?

Mỗi agent có **Approval Rule** — quy định rõ:
- Việc nào agent tự làm (rủi ro thấp).
- Việc nào agent làm xong báo CEO (báo sau).
- Việc nào agent phải hỏi CEO trước (hỏi trước).

Agent không gửi email, không thay giá, không xóa dữ liệu — trừ khi CEO duyệt.

---

## Tôi có thể học nhanh hơn 4 tuần không?

Có, nếu:
- Đã quen terminal/CLI.
- Đã biết cơ bản về AI/prompt.
- Có sẵn danh sách việc lặp.

Có thể hoàn thành trong 2–3 tuần. Nhưng khuyến nghị 4 tuần để làm chắc, không bị "học vội — quên nhanh".
