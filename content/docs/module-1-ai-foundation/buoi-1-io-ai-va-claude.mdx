Sau buổi này, bạn có thể:

- Áp dụng 2 nguyên tắc Responsible AI khi đánh giá output.

## Kiến thức trọng tâm

### 1. LLM hoạt động thế nào?

LLM không phải công cụ tìm kiếm. Nó là mô hình **dự đoán từ tiếp theo** dựa trên xác suất.

> Bạn gõ "Trời hôm nay rất..." → LLM dự đoán "nóng" (70%), "đẹp" (20%), "tối" (5%).

Điều này dẫn đến **hallucination** — LLM sinh ra thông tin nghe hợp lý nhưng sai事实. Vì nó sinh text theo pattern, không phải tra cứu事实.

<Callout>
  <p><strong>Điểm cốt lõi:</strong> LLM sinh text theo pattern xác suất. Khi cần sự thật, <strong>phải kiểm chứng</strong>.</p>
</Callout>

### 2. Ba nền tảng — ba mô hình làm việc

| | Claude | ChatGPT + ChatGPT Work | Hermes Agent |
|---|---|---|---|
| **Bản chất** | Session-based assistant | Session-based + background tasks | Always-on agent 24/7 |
| **Khi nào dùng** | Hỏi-đáp, viết, phân tích | Hỏi-đáp + chạy task nền (tạo file, tìm web) | Tự động hóa việc lặp mỗi ngày |
| **Mạnh nhất** | Chất lượng viết, phân tích dài | Kết nối nhiều tool qua ChatGPT Work | Chạy liên tục qua Telegram, cron, webhook |
| **Yếu điểm** | Dừng khi bạn đóng tab | Phụ thuộc hệ sinh thái OpenAI | Cần setup ban đầu |

<Callout variant="orange">
  <p><strong>ChatGPT Work</strong> là bước tiến mới của OpenAI — cho phép ChatGPT chạy tác vụ nền (tạo file, duyệt web, lên lịch). Nhưng nó vẫn dừng ở "session mở rộng". <strong>Hermes Agent</strong> chạy liên tục 24/7 như một nhân viên online, nhận lệnh qua Telegram, tự ghi nhớ, tự chạy lịch.</p>
</Callout>

## Hands-on lab

### Bước 1 — Tạo tài khoản Claude

1. Vào [claude.ai](https://claude.ai), đăng ký tài khoản.
2. Tour giao diện: chat, Projects, Artifacts.

### Bước 2 — Tạo Project đầu tiên

Project = không gian làm việc có ngữ cảnh dài hạn.

1. Vào **Projects** → **Create Project**.
2. Đặt tên: *"Business Operations Assistant"*.
3. Trong **Custom Instructions**, paste:

<CopySnippets
  snippets={[
    {
      title: 'PROJECT INSTRUCTIONS',
      code: 'Bạn là trợ lý vận hành cho một business nhỏ. Trả lời tiếng Việt, ngắn gọn, thực chiến. Mỗi câu trả lời cần có: kết luận, checklist/bước làm, và bước tiếp theo. Không nói chung chung — luôn gắn vào use case cụ thể.',
    },
  ]}
/>

4. Upload 1–2 file tham khảo (báo giá mẫu, SOP, mô tả sản phẩm).

### Bước 4 — Thử ChatGPT Work (nếu có tài khoản ChatGPT Plus/Team)

1. Vào [chatgpt.com](https://chatgpt.com).
2. Thử giao Task cho ChatGPT Work: *"Tìm giúp anh 3 đối thủ cạnh tranh của [sản phẩm] và tóm tắt điểm khác biệt."*
3. Quan sát: ChatGPT chạy task nền, trả kết quả sau vài phút.
4. So sánh trải nghiệm với Claude.

<Callout>
  <p><strong>Thảo luận:</strong> Claude trả lời ngay, chất lượng cao. ChatGPT Work chạy nền nhưng cần đợi. Hermes Agent chạy nền <strong>luôn luôn</strong> — kể cả khi bạn ngủ.</p>
</Callout>
